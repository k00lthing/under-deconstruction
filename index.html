hidden visuality of the internet

What is a website? At a first glance, this question looks like a very naive, almost trivial one. But in fact, approaching the definition of a website from this perspective may reveal more about its nature. This project explores the nature of a website through its relationship with the browser and exposes hidden architectures in a series of browser extensions.

Extensions

Hide&Seek

This extension’s concept is dead simple - it shows all intentionally hidden elements and hides everything else on the page.

Hide&Seek on GitHub

Field Expedition

By reimagining the landscape of a webpage as an unexplored terrain this extension offers an experience that challenges the conventional web navigation.

Field Expedition on GitHub

Headspace

This extension creates a visual “space” for head element, adding the styles, that render this part of webpage markup visible.

Headspace on GitHub

Borderline Explorer

This extension draws border around every rendered element on the page, revealing their true position and dimensions.

Borderline Explorer on GitHub
